# Engi-Fun | The Missing Teenager 🕵️‍♀️

Welcome to the **Engi-Fun** detective game series! This is an immersive experience that blends interactive mystery-solving with English learning.

## 🎮 About the Game
In this game, you play as a detective. You must read through clues, solve puzzles, and uncover the truth behind the case of the missing teenager. As you progress, you will need to use your English logic to analyze the evidence!
